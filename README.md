# Custom-Ray-Tracer
This is an implementation of a whitted ray tracer utilizing C++, OpenGL, and the GLM library. It currently features:\
-Spheres, Triangles, Planes\
-Directional and point light\
-Motion Blur\
-Depth of Field\
-Glass\
-Metal\
-PBR BRDF\
-Parallelization of rendering\
-Multi-Sampling Anti-Aliasing\
# How to run:
-Write your changes into the scene.txt file with the appropriate info depicted inside of the loadscene function within the renderer.cpp file. Within the scene.txt file are examples of how to format the inputs.
